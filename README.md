## 1. 프로젝트 가져오기  

```sh
git clone https://github.com/eunjisong/Diary.git
```

## 2. 설치하기 

```sh
cd Diary && npm install
```

## 3. 메트로 시작하기

```sh
npm start
```

## 4. 어플 설치하기 

### 안드로이드

```sh
npm run android
```

### 아이폰

```sh
cd ios && bundle install && bundle exec pod install
```

```sh
cd ../ && npm run ios
```
